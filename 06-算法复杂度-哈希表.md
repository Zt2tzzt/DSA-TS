大 O 表示法



顺序查找举例

```typescript
function sequentSearch(array: number[], num: number) {
  const length = array.length
  for (let i = 0; i < length; i++) {
    if (array[i] === num) {
      return i
    }
  }
  return -1
}
```

上面代码，指令执行的次数，大致是是 5n + 2。
