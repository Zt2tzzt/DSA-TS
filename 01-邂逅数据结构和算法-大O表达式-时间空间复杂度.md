常见的数据结构有哪些？



Set/Map 不算数据结构，它们底层用到的 Hash 表，才算数据结构。

像 Java 中的 HashMap、TreeMap，是基于数据结构封装的，本身不是数据结构。底层的 Hash 表，Tree，才是数据结构。
